놀이 방법

  • 😳 / 😌 : 정답 보기 / 숨기기
  • : 정답 직접 쓰기
  • : 숨길 정답을 직접 입력한 후 ‘맞춰봐!’ 버튼을 눌러 보세요. 단어의 자모가 섞입니다.
  • : 미리 입력해 둔 두 글자, 세 글자, 네 글자 단어 중 무작위로 문제가 나옵니다.
  • : 다시 시작 (다 지우기)
  • : 놀이 방법 다시 보기
  • : 소리내서 읽기

  • 한미유치원 어린이들을 위해 만들었습니다. 재미있게 활용해 주세요 !
